# Placeholders

You must have <font color="#1f67ff">[PlaceholderAPI](https://placeholderapi.com/)</font> for these to work!

| Placeholder                   | Description                                      |
|-------------------------------|--------------------------------------------------|
| %\axgens_placed%              | generators placed (player)                       |
| %\axgens_placed%              | generators placed without formatting (player)    |
| %\axgens_limit%               | generator limit (player)                         |
| %\axgens_limit_raw%           | generator limit without formatting (player)      |
| %\axgens_multiplier%          | sell multiplier (player)                         |
| %\axgens_multiplier_raw%      | sell multiplier without formatting (player)      |
| %\axgens_free_slots%          | free generator slots (player)                    |
| %\axgens_free_slots_raw%      | free generator slots without formatting (player) |
| %\axgens_level%               | level of the player (player)                     |
| %\axgens_level_raw%           | level of the player without formatting (player)  |
| %\axgens_loaded%              | generators loaded (global)                       |
| %\axgens_loaded_raw%          | generators loaded without formatting (global)    |
| %\axgens_event_active%        | is any event running, true or false (global)     |
| %\axgens_event_name%          | name of the currently running event (global)     |
| %\axgens_event_next%          | time until next event (global)                   |
| %\axgens_event_next_raw%      | time until next event in seconds (global)        |
| %\axgens_event_time_left%     | event time left (global)                         |
| %\axgens_event_time_left_raw% | event time left in seconds (global)              |
