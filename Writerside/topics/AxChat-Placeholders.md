# Placeholders

- PlaceholderAPI integration:

| Placeholder             | Description                                                                                   |
|-------------------------|-----------------------------------------------------------------------------------------------|
| %\axchat_display_name%  |                                                                                               |
| %\axchat_message_color% | Message color of player (you can change it with the `axchat.messagecolor.#RRGGBB` permission) |
| %\axchat_name_color%    | Name color of player (you can change it with the `axchat.namecolor.#RRGGBB` permission)       |
| %\axchat_server%        | The name of the current server                                                                |
| %\axchat_commandspy%    | Does the player have commandspy enabled? true or false                                        |
| %\axchat_socialspy%     | Does the player have socialspy enabled? true or false                                         |
| %\axchat_msgtoggle%     | Does the player have msgtoggle enabled? true or false                                         |
| %\axchat_staffchat%     | Does the player have staffchat enabled? true or false                                         |
