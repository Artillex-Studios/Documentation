### What is AxHoes?
- An all-in-one harvester hoes plugin focused on performance & customizability.

![image_153.png](image_153.png)

![image_142.png](image_142.png)

## Most Important Features
- Fast: Optimized for both small and large servers.
- Crop regeneration: Crops will automatically regrow after a couple of seconds.
- Packet mode: It is possible to move all the block breaking and setting off the main thread.
- Per player mode: Using packets, players will only see their own crops in the same place.
- Give tool on join: Optionally give every player a tool when the first join.

![image_141.png](image_141.png)
- Tool enchantments: The plugin comes with more than a dozen preconfigured enchantments.

![image_145.png](image_145.png)

![image_146.png](image_146.png)
- Custom enchants: Create your own enchantments using the builtin 15+ triggers.

![image_148.png](image_148.png)
- Leaderboards: Builtin all time leaderboards for all crop types/essence/money/block broken.

![image_144.png](image_144.png)
- Supported crops: By default the plugin comes with 10+ crops, however it is fully configurable, so you can add new ones.

![image_140.png](image_140.png)
- Essence shop: Sell items for essence. Multi page support.

![image_149.png](image_149.png)
- Harvester events: Host events where your players can compete to collect the most crops to receive rewards.

![image_154.png](image_154.png)

![image_155.png](image_155.png)
- Customizable: It is possible change ingame messages, menus, enchants.

![image_150.png](image_150.png)
- Tool levelling: Level up your tool using tool xp. The required xp for all levels can be customized.
- Levelling upgrades: When your tool levels up, you can configure it to change material, you can even add custom textures.

![image_151.png](image_151.png)
- Unlimited levels: The required XP, enchantment effects, prices can be configured using both a math formula and manually.
- Tool prestiges: Prestiging resets the tool stats and gives a permanent boost for the tool. (configurable)

![image_143.png](image_143.png)
- Player xp and levels: Using the builtin level system, players can also gain levels, which other plugins can use.
- Level-up rewards: Give item/command rewards for levelling up.

![image_147.png](image_147.png)
- Supported plugins: You can replace the builtin money/essence/xp system to another supported plugin within seconds.
- Anti nuker: The plugin comes with a simple builtin anti nuker system which prevents blatant cheaters.
- Tool protection options: Protects the player's tool from being lost. (can be turned off)
