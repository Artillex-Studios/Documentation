# Commands

| Command                       | Description           | Permission         | Give By Default |
|-------------------------------|-----------------------|--------------------|-----------------|
| /axafkzone help               | List commands         | axafkzone.help     | OP              |
| /axafkzone reload             | Reload plugin         | axafkzone.reload   | OP              |
| /axafkzone wand               | Get selection wand    | axafkzone.wand     | OP              |
| /axafkzone create &lt;zone>   | Create zone           | axafkzone.create   | OP              |
| /axafkzone delete &lt;zone>   | Delete zone           | axafkzone.delete   | OP              |
| /axafkzone redefine &lt;zone> | Redefine area of zone | axafkzone.redefine | OP              |
| /axafkzone tp &lt;zone>       | Teleport to zone      | axafkzone.tp       | OP              |
