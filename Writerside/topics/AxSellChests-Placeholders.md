# Placeholders

| Placeholder                     | Description                                        |
|---------------------------------|----------------------------------------------------|
| %\axsellchests_place_limit%     | Number of maximum sell chest slots                 |
| %\axsellchests_placed%          | Number of free sell chest slots                    |
| %\axsellchests_free%            | Number of free sell chest slots                    |
| %\axsellchests_place_limit_raw% | Number of maximum sell chest slots (not formatted) |
| %\axsellchests_placed_raw%      | The number of placed sell chests (not formatted)   |
| %\axsellchests_free_raw%        | Number of free sell chest slots (not formatted)    |
