# Commands

| Command                                          | Permission                                       | Default  | Description            |
|--------------------------------------------------|--------------------------------------------------|----------|------------------------|
| /axpickaxes [help]                               | axtools.help                                     | true     | Help                   |
| /axpickaxes give &lt;player> [level]             | axtools.give                                     | OP       | Give pickaxe to player |
| /axpickaxes tool setlevel &lt;level> [reset-xp]  | axtools.tool.setlevel                            | OP       | Set tool level         |
| /axpickaxes tool setxp &lt;xp>                   | axtools.tool.setxp                               | OP       | Set tool XP            |
| /axpickaxes tool setprestige &lt;prestige>       | axtools.tool.setprestige                         | OP       | Set tool prestige      |
| /axpickaxes reload                               | axtools.reload                                   | OP       | Reload configuration   |
| /axpickaxes essence view [player]                | axtools.essence.view, axtools.essence.view.other | true, OP | View essence balance   |
| /axpickaxes essence give &lt;player> &lt;amount> | axtools.essence.give                             | OP       | Give essence           |
| /axpickaxes essence take &lt;player> &lt;amount> | axtools.essence.take                             | OP       | Take essence           |
| /axpickaxes essence set &lt;player> &lt;amount>  | axtools.essence.set                              | OP       | Set essence            |
| /axpickaxes essence reset [player]               | axtools.essence.reset                            | OP       | Reset essence          |
| /axpickaxes essence pay &lt;player> &lt;amount>  | axtools.essence.pay                              | true     | Send essence to player |
| /axpickaxes xp view [player]                     | axtools.xp.view, axtools.xp.view.other           | true, OP | View player xp balance |
| /axpickaxes xp give &lt;player> &lt;amount>      | axtools.xp.give                                  | OP       | Give player xp         |
| /axpickaxes xp take &lt;player> &lt;amount>      | axtools.xp.take                                  | OP       | Take player xp         |
| /axpickaxes xp set &lt;player> &lt;amount>       | axtools.xp.set                                   | OP       | Set player xp          |
| /axpickaxes xp reset [player]                    | axtools.xp.reset                                 | OP       | Reset player xp        |
| /axpickaxes xp reset [player]                    | axtools.xp.reset                                 | OP       | Reset player xp        |
| /axpickaxes shop [player]                        | axtools.shop, axtools.shop.other                 | true, OP | Open shop gui          |
| /axpickaxes leaderboard [player]                 | axtools.leaderboard, axtools.leaderboard.other   | true, OP | Open leaderboard gui   |

> Note that the permissions start with `axtools`, not axpickaxes!
