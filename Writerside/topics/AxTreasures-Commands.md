# Commands

| Command                                 | Permission    | Given by Default |
|-----------------------------------------|---------------|------------------|
| /axtreasures help                       | axchat.help   | op               |
| /axtreasures create [cooldown]* [title] | axchat.create | op               |
| /axtreasures remove                     | axchat.remove | op               |
| /axtreasures reset &lt;player>          | axchat.reset  | op               |
| /axtreasures reload                     | axchat.reload | op               |

\* The cooldown can be any short time value, like: 5s, 10m5s or you can also use only numbers for seconds.
