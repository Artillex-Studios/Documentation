> You can read all the upcoming features [here](AxAuctions-V2-Update.md)!
{style="warning"}

* A feature rich auction house plugin with multi-server support.

![image_220.png](image_220.png)

![image_189.png](image_189.png)

### Features:
- H2 and MySQL databases supported.
- Multi-Server Support: SQL, (zero setup required) Proxy, and for maximum speed Redis and RabbitMQ are also supported.
- Folia Support
- Multi Currency Support: If multiple currencies are enabled, a currency selector will be opened where players can pick what to sell their item for.

![image_195.png](image_195.png)

![image_196.png](image_196.png)
- Placeholder Currencies: While more than a dozen currencies are supported by default, you can create new ones simply by using placeholders.
- Purchase Confirmation: The plugin asks the player to confirm their purchase to avoid unwanted purchases.

![image_203.png](image_203.png)

![image_200.png](image_200.png)
- My Items Page: You can view or cancel your own items in a separate gui.

![image_197.png](image_197.png)
- Shulker Content Viewer: Right click on shulkers in any menu of the plugin to see all the contents of the shulker.

![image_201.png](image_201.png)

![image_202.png](image_202.png)
- Fast & Optimized: The plugins utilizes async tasks, and it loads items lazily to support 100s of pages full of items with no lag.
- Discord Webhooks: Send out a webhook to your discord server when new auctions are created or when items are bought.

![image_219.png](image_219.png)
- Item Name Translations: The plugin can display the name of items in the selected language in messages.
- Log Viewer: The plugin tracks all actions made by players/staff and groups all this information into an easy-to-use menu.

![image_207.png](image_207.png)

![image_208.png](image_208.png)
- History Viewer: See all the previously sold items on the server in a compact and fast menu.

![image_211.png](image_211.png)

![image_212.png](image_212.png)
- Deleted Item Viewer: Forgot your item in the auction house and it got deleted? It is possible to view all the deleted items through a simple menu.

![image_205.png](image_205.png)

![image_206.png](image_206.png)
- Works Out of The Box: The plugin comes with key features enabled so you can get started using it without setup. The extra features can be turned on easily.
- Highly Customizable: You can change all the messages and most feature offer settings that you can use to tune them for your server.

![image_216.png](image_216.png)
- Category Viewer: The plugin provides a full category system which can be enabled to make it easier for players to find items.

![image_190.png](image_190.png)

![image_191.png](image_191.png)

![image_214.png](image_214.png)

![image_192.png](image_192.png)
- Smart Search: Search for items by their seller, item name, lore, material, enchantments. The plugin supports sign, chat and anvil menu search.

![image_198.png](image_198.png)

![image_199.png](image_199.png)
- Item Sorting: Players can pick to sort items by newest, oldest or by the item's price.

![image_194.png](image_194.png)
- Statistics Tracking: The plugin tracks various stats like items sold, items purchased, money made, money spent.

![image_204.png](image_204.png)
- Item Filtering: Players can filter items by currency or category.

![image_193.png](image_193.png)
- Auction Limits: Set a limit by permission or command to limit how many items can players have in the auction house at once.

![image_215.png](image_215.png)
- Notifications: The plugin notifies players when their item is bought, expires or when it gets deleted.

![image_221.png](image_221.png)
- Offline Player Support: If the seller is offline, the plugin will remember and give their earnings and send their notifications next time they join.
- Admin Utilities: Staff can cancel or delete items from the auction house. They can also recover sold or deleted items.

![image_209.png](image_209.png)

![image_210.png](image_210.png)
- Item Blacklisting: The plugin provides an advanced blacklist system that can be used to block unwanted or large items off the auction house.

![image_213.png](image_213.png)
- Tax System: You can set a tax percentage for each currency to prevent your server's economy from inflating.

![image_217.png](image_217.png)

- Number Formatting And Abbreviations: You can use values like 100k, 100m, 1b to sell items for a lot of money quickly.

![image_218.png](image_218.png)
